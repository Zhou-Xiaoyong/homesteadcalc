How It Works

Chicken coop size requirements are based on animal welfare standards and the birds' need for space to roost, nest, and move comfortably. Overcrowding leads to stress, aggression, and disease.

Space Requirements by Breed Size

Larger breeds need more space per bird:

  • Standard/Large (8-10 lbs): 4 sq ft/bird in coop, 10 sq ft/bird in run
  • Medium (5-7 lbs): 3 sq ft/bird in coop, 8 sq ft/bird in run
  • Bantam/Small: 2 sq ft/bird in coop, 5 sq ft/bird in run
  • Mixed Flock: Uses weighted average based on your flock composition

Housing Style Impact

  • Confined: Birds stay in coop+run full time. Requires full space calculations.
  • Free-range: Birds roam during day, cooped at night. Run space reduced by 50%.
  • Pasture rotation: Portable coop moved regularly. Minimum coop space still required.
  • Deep litter: Bedding builds up over time. Requires 6-8" depth and extra ventilation.

Nesting Boxes and Roosting

  • Nesting boxes: 1 box per 4-5 hens, each 12"×12"×12"
  • Roosting bars: 8-10 linear inches per bird (standard), 6-8" (bantam)
Coop area (sq ft) = Number of chickens × Coop sq ft per bird Run area (sq ft) = Number of chickens × Run sq ft per bird (if free-range: Run area × 0.5) Nesting boxes = Ceiling(Number of chickens / 5) Roosting bar length (inches) = Number of chickens × Inches per bird (Roosting bar length in feet = Roosting bar length in inches / 12) Total footprint (sq ft) = Coop area + Run area + Storage (16 if selected) + Brooder (16 if selected) Note: Suggested dimensions are approximate squares/rectangles that meet the minimum area requirements.

Space Requirements Quick Reference

  • Standard: 4 sq ft coop, 10 sq ft run
  • Medium: 3 sq ft coop, 8 sq ft run
  • Bantam: 2 sq ft coop, 5 sq ft run
  • Nest boxes: 1 per 4-5 hens
  • Roost: 8-10" per bird

Tips for Coop Design

  • Face coop door away from prevailing winds
  • Provide 1-2 windows for light and ventilation
  • Elevate coop 12-18" off ground for drainage
  • Hardware cloth (not chicken wire) for predator protection
  • Plan for easy cleaning access
